## 🕹️ Controles del Juego

* **Movimiento:** Usa las **Flechas direccionales** o las teclas **W, A, S, D** para mover tu nave.
* **Disparar:** Presiona la **Barra Espaciadora** para lanzar misiles.
* **Objetivo:** Sobrevive y destruye a la nave enemiga. Evita los misiles rivales (tu nave solo resiste 2 impactos).

## 🚀 Cómo abrir y probar el proyecto en Unity

Si quieres ver el código, modificar el juego o probarlo directamente en el motor, sigue estos pasos:

1. **Descargar el proyecto:**
   * Puedes clonar este repositorio usando Git: `git clone [URL_DE_TU_REPOSITORIO]`
   * O puedes descargar el proyecto como un archivo ZIP desde el botón verde "Code" > "Download ZIP" en GitHub y descomprimirlo en tu computadora.

2. **Añadir a Unity Hub:**
   * Abre **Unity Hub**.
   * Ve a la pestaña **Projects** y haz clic en el botón **Add** (o *Open*).
   * Busca y selecciona la carpeta raíz del proyecto (la carpeta que descargaste).

3. **Abrir el Editor:**
   * Haz clic en el proyecto recién añadido en Unity Hub. *(Nota: Es recomendable usar la misma versión de Unity con la que se creó el proyecto para evitar errores de compatibilidad).*

4. **Cargar la Escena:**
   * Una vez que Unity abra, ve a la ventana de **Project**.
   * Navega a la carpeta `Assets` > `Scenes`.
   * Haz doble clic en tu escena "SampleScene" principal para cargar el nivel.

5. **¡A jugar!:**
   * Haz clic en el botón de **Play** (▶️) en la parte superior central del editor para iniciar el juego directamente en la pestaña *Game*.
